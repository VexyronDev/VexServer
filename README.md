# VexServer

Roblox Nametag System mit eigenem Server.

## Features

- Zeige Nametags über Spieler-Köpfen
- Setze Teams mit Farben
- Server-gesteuert über Node.js

## Befehle

```lua
setTeam("Vex")     -- Pink
setTeam("Hacker")  -- Rot
setTeam("Admin")   -- Grün
setTeam("VIP")     -- Gold
setTeam("Owner")   -- Orange
setTeam("Mod")     -- Blau
setTeam("")        -- Kein Team
